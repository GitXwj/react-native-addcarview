# react-native-addcarview
一个简单实用的实现添加购物车动画的控件

- 安装:`npm install react-native-addcarview --save` or `yarn add react-native-addcarview`
- 使用: 具体用法见example
- 注意点:动画开始结束位置是通过传入组件调用measureInWindow方法测量得到的，请确保传入控件有measureInWindow方法。如果没有这个方法（大部分组件有），也可以传入坐标

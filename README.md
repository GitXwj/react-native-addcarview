# react-native-addcarview
一个简单实用的实现添加购物车动画的控件

- 安装:
   ```
       npm install react-native-addcarview --save
   ```
- 使用:
   ```
       import {ShoppingCarView, startAddShopAnim} from 'react-native-addcarview'
   ```
